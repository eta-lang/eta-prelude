Description

The Num type class defines basic numerical operations:

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Negation (negate)
  • Absolute value (absoluteValue)
  • Sign of a number (numberSign)
  • Conversion from Integer (fromInteger)
