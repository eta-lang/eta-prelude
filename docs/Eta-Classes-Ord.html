Eta.Classes.Ord

Description

The Ord class defines ordering

A helper datatype exists to denote the Ordering

>>> compare 1 2
LT
>>> compare 2 1
GT
>>> compare 2 2
EQ
>>> 1 < 2
True
>>> 1 > 2
False
>>> 1 >= 2
False
>>> 1 <= 2
True
